Form Project
FQ: How can we implement the form elements we have learned to make a professional sign up form?

Overview: We will use all the skills we have learned with Form tags to create a custom form of your choice. 

You will need to:
Think of a form idea. It could be anything that collects data: 
	college interest form
	pizza ordering form, 
	video game related form, 
	anything that has all the requirements below.
Begin to implement the structure of your form using Replit.

Requirements: 
Your form should be properly formatted with the following tags:
<form>
<fieldset> (when applicable), at least once
<legend> (when applicable), at least once
<label>
<input>
type=”text”
type=”password”
type=”email”
pattern attribute (on at least one input)
type=”radio”
name attribute 
type=”checkbox”
name attribute
type=”submit”
required attribute
datalist tag w/ input list=”” attribute

Your form should be visually appealing with CSS and should match the design provided.


On submit of the form, Create a javaScript function to use event.preventDefault() and display an alert that form submission was successful.
